# prime-search

This is a prime-searching project.

## How does it work?

It uses the Proth's theroem to check primality.

## How to install

```sh
# Install Anaconda on Linux
pip install tqdm gmpy2
git clone https://github.com/StevenJinyanCheng/prime-search.git
cd prime-search
python proth.py -h
```
